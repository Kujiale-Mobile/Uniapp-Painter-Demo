# Uniapp-Painter-Demo

本项目展示了如何在 Uniapp 框架中使用 Painter

## 如何运行

```
git clone https://github.com/Kujiale-Mobile/Uniapp-Painter-Demo.git
```

代码克隆后，使用 HBuilder X打开即可运行项目。
